QuickBook Backend API

Esta es la API de backend para la aplicación QuickBook. Para usar la API, accede a alguno de los endpoints disponibles:

Tests: /api/tests - Gestión de tests (GET, POST)
Imagen: /api/images?id=imagen_id - Obtener imágenes de Airtable
Diagnóstico: /api/airtable-check - Verificar conexión con Airtable

Páginas de Prueba

Carga de Imágenes: /image-upload-debug.html - Prueba para cargar imágenes en Airtable
Creación de Tests: /test-upload-example.html - Prueba para crear tests completos con imágenes

Para más información, consulta la documentación o contacta al administrador.
